 # 操作说明
 wsad键 -- 人物移动，双击可进行奔跑，默认状态下添加了八方向移动<br/>
 E键 -- 捡起武器 （通过触发和射线同时检测）<br/>
 F键 -- 扔掉武器<br/>
 R键 -- 换弹<br/>
 鼠标左键 -- 射击<br/>
 长按鼠标右键 -- 进行瞄准<br/>
 X键 -- 切换武器的射击模式<br/>
 Tab键 -- 视角切换<br/>
 Space键 -- 跳跃<br/>
 1 键 -- 如果两把武器，此键切换<br/>
